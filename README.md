#### Bun

```
bun install
bun run dev
```

```
open http://localhost:3000
```

#### Nodejs

Set the correct nodejs version, run

```
nvm use
```

Install dependencies and run the project

```
pnpm install
pnpm run dev:node
```

```
open http://localhost:3000
```

Dependencies

```
ts-node-dev,
@hono/node-server
```
